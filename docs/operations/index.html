Operations

Title#

Title of the embed

```embed
title: {content}
```

Description#

```embed
description: {content}
```

Colour#

```embed
color: {hex}
```

Fields#

```embed
{name}: {value}
```
tip

If {name} shares the same name as another function, put a \ infront of it. You can also insert {inline} after the value to make the field inline.

Title Url#

```embed
url: {link}
```

Display Name#

```embed
displayname: {name}
```

Thumbnail Image#

```embed
thumnail: {image_url}
```

Timestamp#

```embed
timestamp: true
```

Avatar Image#

```embed
avatarurl: {image_url}
```
